Tags HTML e Propriedades CSS

HTML

Tag div (Divisão):

É um elemento de contêiner usado para agrupar e estruturar conteúdo em uma página da web. Frequentemente é utilizada com CSS para aplicar estilos ou propriedades de layout.

Saiba mais.

Tag a (Âncora):

É usada para criar hiperlinks, permitindo que os usuários naveguem para outras páginas ou recursos. É comumente usada em combinação com o atributo href para especificar o destino.

Saiba mais.

Tag img (Imagem):

É usada para incorporar imagens em uma página da web. Ela requer o atributo src para especificar a URL do arquivo de imagem e o atributo alt para texto alternativo.

Saiba mais.

Tag ul (Lista não ordenada) e ol ((Lista ordenada):

Essas tags são usadas para criar listas. ul é para listas não ordenadas (com marcadores), e ol é para listas ordenadas (numeradas). Elas contêm elementos li (itens da lista).

Saiba mais.

CSS

Propriedade color:

é usada para definir a cor do texto de um elemento. Pode receber valores em diversos formatos, como cores nomeadas (por exemplo, "red") ou códigos hexadecimais (por exemplo, "#FF0000").

Saiba mais.

Propriedade font-size:

Define o tamanho do texto dentro de um elemento. Pode ser especificada em diferentes unidades, como pixels, ems ou porcentagens.

Saiba mais.

Propriedade margin:

Define o espaço fora de um elemento. Pode ser usada para criar espaço ao redor de um elemento, separando-o de outros elementos na página. Os valores podem ser especificados para margens superior, direita, inferior e esquerda.

Saiba mais.

Propriedade padding

Define o espaço entre o conteúdo de um elemento e sua borda. É usada para controlar o espaçamento interno dentro de um elemento. Assim como margin, os valores podem ser definidos para preenchimento superior, direito, inferior e esquerdo.

Saiba mais.
